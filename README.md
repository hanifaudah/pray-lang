## How To Use
1. Download `pray_compiler.zip` from the latest release
2. Extract the compiler
3. Write code with `.рЯЩП` extension
4. Compile the code code
```
./pray_compiler [input_file.рЯЩП] [output_file]
```
5. Run binary
```
./output_file
```
